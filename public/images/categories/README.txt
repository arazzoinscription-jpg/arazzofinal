Images de catégories de la boutique de patrons.

Dépose ici une image par type de vêtement (format portrait 4:5 conseillé, .jpg).
Le nom du fichier DOIT correspondre au slug du type :

  robe.jpg
  jupe.jpg
  pantalon.jpg
  haut.jpg
  veste.jpg
  ensemble.jpg
  traditionnel.jpg
  accessoire.jpg

Tant qu'un fichier est absent, la vignette affiche automatiquement un dégradé
de marque + le nom de la catégorie (aucune erreur). Dès que l'image est déposée,
elle remplace le dégradé sans aucune modification de code.

Ces images peuvent être générées avec Higgsfield (IA) une fois le compte crédité.
